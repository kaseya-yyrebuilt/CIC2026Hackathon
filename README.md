# ProofHire MVP

ProofHire is a hackathon MVP for verified reference letters and work experience credentials.

## Demo Accounts

- Candidate: `candidate@test.com`
- Referrer: `referrer@test.com`
- Employer: `employer@test.com`

## Primary Demo Flow

1. Continue as Employer.
2. Select the Software Engineer job.
3. Open Candidate Results.
4. View Maya Chen's Verification Profile.
5. View the original reference letter.
6. Click AI Summary to generate the job-fit summary.
